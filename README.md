# Perch — an envelope, not a verdict

Single-file pacing & symptom tracker. Data lives in your device's browser
storage (localStorage) — nothing is sent anywhere.

## Hosting (GitHub Pages)
1. Upload all files in this folder to a GitHub repository.
2. Repo Settings → Pages → Source: "Deploy from a branch" → main → / (root) → Save.
3. Your app: https://YOURNAME.github.io/REPONAME/

## iPhone install
Open the URL in Safari → Share → Add to Home Screen → keep "Open as Web App" ON.

## Swap the icon
Prefer the geometric mark? Replace icon-180.png, icon-192.png and icon-512.png
with the files from alt-icons/ (same names).

## Restore your data
Settings → Paste JSON… → paste your full Perch export (settings + days come across).
